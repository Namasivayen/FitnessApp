# FitnessApp Monorepo

This monorepo contains the FitnessApp Progressive Web Application and supporting services.

## Structure

- `apps/web` – Next.js PWA (TypeScript)
- `apps/api` – Node.js + Express API (TypeScript)
- `packages/shared` – Shared types and constants
- `docs` – Architecture and decisions

## Getting Started

- Install dependencies: `npm install`
- Each app/package can be developed and built independently.

---

See `/docs` for architecture and design decisions.
